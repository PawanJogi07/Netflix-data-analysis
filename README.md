## Dataset Information

This project uses a Netflix dataset containing detailed information about movies and TV shows available on the platform.

### Key Features:
- **Type:** Movie or TV Show
- **Title:** Name of the content
- **Country:** Production country
- **Release Year:** Year of release
- **Rating:** Age certification (e.g., PG, TV-MA)
- **Duration:** Length of movie or number of seasons
- **Genre:** Category of content

### Data Processing:
- Handled missing values using data cleaning techniques
- Filtered relevant columns for analysis
- Prepared dataset for exploratory data analysis (EDA)

The dataset was used to analyze content distribution, trends, and patterns in Netflix's library.
